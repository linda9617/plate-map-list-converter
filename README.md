# plate-map-list-converter

Plate Map ↔ List Converter is a lightweight web tool for bi-directional conversion between microplate map format and well-list format.

It supports 96-, 384-, and 1536-well layouts, automatically detects input structure, and helps users quickly clean, transform, and export plate data for downstream analysis. With editable spreadsheet-style input and instant visual plate preview, it is designed to make routine lab data formatting faster and more reliable.
